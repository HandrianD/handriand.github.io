About

RADIUSdesk is a decade old and still relevant!
We started in 2012 with an enhanced GUI to FreeRADIUS. MESHdesk was added soon afterwards as an easy GUI for Batman-Adv based mesh networks. APdesk completed the package to centally manage APs.
It has a modern architecture with the GUI and back-end seperate. This means everyting can be done using API calls.
Over the years we kept up with technology and now manage hardware with up to three Wi-Fi radios using OpenWrt 21.02.x.
